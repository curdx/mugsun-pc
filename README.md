# mugsun-pc

Mugsun 平台 PC 管理端 —— Vue 3 + Vite + TypeScript + Element Plus。

## 环境要求

- Node.js >= 20.19
- pnpm >= 8.8

## 开发

```bash
pnpm install
pnpm dev
```

## 构建

```bash
pnpm build
```
